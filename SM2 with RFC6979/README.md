实现签名的关键  
(x′1; y′1)  
= [s′]G + [t]PA  
= [s′]G + [s′] PA + [r′] PA  
= [s′]G +[s′][dA] G +[r′·dA] G  
= [(1+ dA)×s′] G+[r′·dA] G  
= [k − r · dA) ] G+[r′·dA] G  
= [k] G   
= ( x1, y1 )
